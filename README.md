# Project Title
Password Generator

## Overview
This project is a Python-based password generator that creates secure and customizable passwords based on user preferences. It uses secure randomization for strong password generation and provides strength evaluation.

## Features
- Generate passwords with user-defined length  
- Option to include uppercase letters, digits, and special characters  
- Password strength evaluation (Weak, Moderate, Strong)  
- Secure random selection using Pythonâ€™s `secrets` module  

## Technologies Used
- Python 3.x  
- Python `string` and `secrets` modules
